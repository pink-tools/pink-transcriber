- `pink-transcriber-darwin-arm64` - macOS ARM64
- `pink-transcriber-linux-amd64` - Linux x64
- `pink-transcriber-windows-amd64.exe` - Windows x64

On first run, downloads [pink-whisper](https://github.com/pink-tools/pink-whisper) + model (~4GB).

**Structure:**

```
~/pink-tools/
├── pink-transcriber
└── pink-whisper/
    ├── pink-whisper
    └── ggml-large-v3.bin
```
